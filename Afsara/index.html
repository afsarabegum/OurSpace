Web Media Production BA(Hons)

At Web Media you will learn how to create professional web content and services. You will take part in both practical workshops and theoretical lessons, taught by experienced industry leaders, and guests from companies such as Google, Mozilla, Burberry and Code Club.

Our graduates start their careers as Web designers, app developers, UX designers, digital project managers, UI designers, social media producers and game designers among others. They work both within agencies and as freelancers.
